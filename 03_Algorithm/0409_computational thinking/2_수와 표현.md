## 6번

![수와표현_5](2_수와 표현.assets/수와표현_5.PNG)

`sol`

f(x) = 3 log(x+3) + 1

f(x)-1 = 3log(x+3)

(f(x)-1) / 3 = log(x+3)

2^((f(x)-1)/3) = x + 3

x = 2^((f(x)-1)/3) - 3
