##### isalpha, isdigit 

```python
password = 'abcde'

flag_alpha = False
flag_number = False

for i in password:
    if i.isalpha():
        flag_alpha = True
    if i.isdigit():
        flag_number = True
if not flag_alpha:
    print("비밀번호에 알파벳이 사용되지 않았음")
elif not flag_number:
    print("비밀번호에 숫자가 사용되지 않았음")
else:
    print("완벽한 비밀번호이다.")
```

##### find , index 차이

```python
line2 = "안녕하시요"
print(line2.find("녕"))
print(line2.index("녕"))

# 1
# 1
```

##### 문자열 숫자를 정수로 변환하기

```python
def atoi(num_str):
    value = 0  # 최종 값을 담을 변수
    for i in range(len(num_str)):
        value *= 10
        value += ord(num_str[i]) - ord('0')
    return value
        
num_str = "1234"
num_int = atoi(num_str)
print(num_int, type(num_int))
```
