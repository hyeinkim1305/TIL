# todo-list-vuex-project

### 메인 흐름
1. method의 컴퍼넌트에서 dispatch를 활용해 actions를 호출
2. actions에 정의된 메서드는 commit을 활용해 mutations를 호출
3. mutation에 정의된 메서드는 state를 조작한다.
